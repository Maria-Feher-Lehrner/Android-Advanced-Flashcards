***LEHRVERANSTALTUNG ANDROID ADVANCED (WiSe 2024/25)***
**FLASHCARDS LERNAPP**

Die App erfüllt die Basisfunktionalität einer digitalen Flashcard-App mit folgenden Features:
- initiales Set an Vokabeln (via CSV file)
- erweiterbar um eigene Vokalbeln (Wortpaare)
- Check, ob neuer Eintrag bereits existiert - inkl. Feedback
- Bearbeitbarkeit der einzelnen Wortpaare (update & delete)
- Bearbeitbarkeit der gesamten Liste (delete), wobei die initiale Liste sofort wieder eingespielt wird, sobald die Datenbank leer ist
- Quizfunktion in beide Sprachrichtungen
- Randomisierte Auswahl von Vokabeln aus der Liste
- Trackingmöglichkeit ob Vokabel gewusst wurde oder nicht (Lernlevel)
- Gratulationsnotification sobald das Lernlevel 3 erreicht

Implementierte Blöcke aus Aufgabenstellung:

+ Base: Project Setup and ViewModels
+ Base: Repository and Navitation
+ Block 1: LiveData and Livecycle Components
+ Block 2: SQL with Google Room
- Block 3: API Access and Serialization with ktor (started the block but didn't finish implementing it)
+ Block 4: Dependency Injection with Koin
+ Block 5: Dependency Injection with Koin (Google Room)

No further Blocks implemented.
